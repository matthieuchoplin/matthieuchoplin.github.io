Screen

Matthieu Choplin

2014-06-30 13:44

Source

Install screen

sudo apt-get install screen

Start a new screen session with session name

screen -S <script_name>

Attach to a running session with name

screen -r <script_name>

Escape key

Ctrl-a and d
